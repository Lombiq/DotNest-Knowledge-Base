# Language packages



The following language packages are available on all DotNest sites for every built-in theme and module of Orchard:

- Chinese Simplified (zh-CN, 100%)
- Dutch (nl-NL, 85%)
- French (fr-FR, 100%)
- Hungarian (hu-HU, 100%)
- Italian (it-IT, 100%)
- Russian (ru-RU, 77%)
- Spanish (es-ES, 100%)
